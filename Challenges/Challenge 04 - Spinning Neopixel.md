# Challenge 04 - Spinning Neopixel
This challenge explores using the built-in Neopixels.
## Mild 🌶️
The Neopixels can be individually controlled.

* Start with pixel 0 on and the rest off.
* Wait a short amount of time, and turn pixel 1 on and the rest off.
* Continue until pixel 9 is on and the rest are off.

## Medium 🌶️🌶️
The Neopixels can be individually controlled.

* Have a single Neopixel of your favorite color continuously spin counter-clockwise.
* It should start at position 0.
* It should not stop once it gets to position 9, but continue spinning around.

## Spicy 🌶️🌶️🌶️
The Neopixels can be individually controlled.

* Have each Neopixel set to a different color of the rainbow.
* Have the rainbow of Neopixels continuously spin counter-clockwise.
